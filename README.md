Instalation & requirements
---

* Requires PyGame
* Download the source
    ```shell
        git clone https://github.com/Data5tream/platformer.git
    ```
* Run the platformer.py script within the platformer folder
    ```shell
        ./platformer.py
    ```
    or
    ```shell
        python platformer.py
    ```
